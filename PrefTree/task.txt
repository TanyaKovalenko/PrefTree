Необходимо построить сеть, состоящую из узлов, соответствующих фонемам (список приведен в приложенном файле phone.txt, 
его можно не грузить, а просто вшить в код, это не очень важно), таким образом, чтобы при прохождении через эту сеть генерировались все слова из словаря (приложенный файлик dictionary.txt).

Формат словаря: слово=фонемы#идентификатор

Нужно реализовать следующие функции:
1) Создание сети по словарю из файла (например по файлу dictionary.txt);
2) Получение идентификатора слова по его произношению (по набору фонем) или сообщение, что слова с таким произношением в словаре нет;
3) Восстановление из сети словаря вида: фонемы#идентификатор, для сравнения с оригиналом.

Сеть должна начинаться с виртуального узла начало (к примеру, тип NODE_TYPE_BEGIN) и заканчиваться произвольным количеством узлов. 
Суммарное количество узлов, соответствующих фонемам (тип NODE_TYPE_PHONE) должно быть по возможности минимальным. 
При прохождении сети слева направо, от начального узла к конечным, по всем возможным узлам и переходам сети, на выходе должен получиться исходный словарь, 
причем лишних слов в словаре появиться не должно. При прохождении по каждому слову, должен встретиться только один виртуальный узел, содержащий идентификатор данного слова (тип NODE_TYPE_WORD).

Например, для вот такого словаря: 
машина - 1 
матрос - 2
маша - 3

должна генерироваться сеть из файла net.png (для наглядности использованы буквы, а в сетках должны быть идентификаторы фонем).
